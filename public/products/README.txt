PRODUCT IMAGES — DROP FILES HERE
================================

Place the 7 product photos in this folder with EXACTLY these filenames:

  gongura.jpg         → Gongura Pickle photo
  avakaya.jpg         → Mango Avakaya (Avakaya Pickle) photo
  mysorepak.jpg       → Mysore Pak photo
  sunnunda.jpg        → Sunnunda (Palli Undalu) photo
  kandipodi.jpg       → Kandi Podi photo
  karvepakupodi.jpg   → Karvepaku Podi photo
  mixture.jpg         → Mixture (Snacks) photo

After dropping them here, rebuild the frontend (npm run build) or
the dev server will pick them up automatically.

These filenames are already wired up in:
